# Changelog — 2026-03-01 05:45

## Admin-configurable site settings, clean up public header

- **Removed navbar from public view:** Non-logged-in visitors no longer see the full navbar (no duplicate Astra branding, no redundant Projects link). Instead, a small "Log In" button floats in the top right corner.
- **Admin site settings:** New `site_settings` table (migration 008) stores key-value pairs. Admin can configure the landing page tagline and footer copyright text from Admin > Site Settings.
- **Configurable tagline:** The landing page subtitle (default: "Bug Tracker — MiraNova Studios") is now pulled from the `tagline` setting.
- **Configurable footer:** The footer copyright text (default: "© 2026 MiraNova Studios") is now pulled from the `footer_text` setting.
- Settings are loaded into all views via the locals middleware as `siteSettings`.
