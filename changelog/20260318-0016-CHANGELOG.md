# Changelog — 2026-03-18

## Added
- **Show on Summary** flag for projects (checkbox on Manage Projects form) to hide individual projects from the Projects Summary page.
- **Show on Summary** toggle for categories (button on Manage Categories page) to hide all projects in a category from the summary.
- Migration 028 adds `show_on_summary` column to both `projects` and `categories` tables (default 1).
- Projects Summary query now filters out projects where `show_on_summary = 0` or whose category has `show_on_summary = 0`.
