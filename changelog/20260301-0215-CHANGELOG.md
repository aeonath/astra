# Changelog — 2026-03-01 02:15

## Admin password reset restricted to CLI only

- Admin users can no longer have their password reset via the web UI. The reset form is hidden for admin accounts and the route rejects the request with a message pointing to `npm run reset-password`.
- When a regular user's password is reset via the web UI, their active sessions are destroyed so they are forced to log in with the new password.
