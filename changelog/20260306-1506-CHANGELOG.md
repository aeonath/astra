# Changelog — 2026-03-06 15:06

## Clear project tabs on user change

- Header injects current user ID via <meta name="astra-user-id"> when logged in
- project-tabs.js compares stored user ID (astra_user_id) against current on every page load
- If user has changed (login/logout/switch), astra_open_tabs is cleared and astra_user_id is updated
- Prevents one user's open tabs from appearing in another user's session
