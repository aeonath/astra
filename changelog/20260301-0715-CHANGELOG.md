# Changelog — 2026-03-01 07:15

## Added: Public Bug Report & Feature Request Submissions

- Public visitors can now submit bug reports and feature requests without logging in
- Landing page shows "Report a Bug" and "Request a Feature" buttons at the bottom (centered, above footer)
- Submission form includes: name, email (optional), category filter, project selection, summary, and details
- Category dropdown dynamically filters the project list
- Form clearly states submissions go to the dev team and won't appear on the site directly
- Submissions stored in new `public_submissions` table (migration 012)
- Admin dashboard shows "New Submissions" count linking to the submissions inbox
- Admin submissions page lists all submissions with type, project, submitter, summary, status, and date
- Admins can mark submissions as reviewed or dismissed
