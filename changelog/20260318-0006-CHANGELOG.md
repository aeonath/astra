# Changelog — 2026-03-18

## Changed
- Separated archive/reactivate from the Active flag. A new `archived` column (migration 025) now controls whether a project is hidden everywhere except the Manage Projects list.
- The `active` flag now exclusively controls Projects Summary visibility.
- All project queries across routes (landing, projects, bugs, submissions, summary) now filter on `archived = 0` instead of `active = 1`.
- Archive/Reactivate toggle updated to use the `archived` column in both admin and manage-projects routes.
- Manage Projects table now shows archived status from the `archived` column.
