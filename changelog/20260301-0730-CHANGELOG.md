# Changelog — 2026-03-01 07:30

## Changed: AJAX-based project dropdown on public submission form

- Project dropdown is now empty and disabled until a category is selected
- When a category is selected, projects are fetched via AJAX from `/projects/submit/projects?category_id=N`
- Category placeholder changed from "All Categories" to "Select a category"
- Removed pre-loaded project options from the template (no longer passed from route handler)
