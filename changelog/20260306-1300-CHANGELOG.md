# Changelog — 2026-03-06 13:00

## Separate inline edit for Description and Additional Notes cards

- Edit link is now inside each gray card (bottom-right corner) rather than in the heading row
- Description and Additional Notes each have their own independent edit form that swaps in place
- Each form sends hidden fields for the values it is not editing so the other fields are preserved
- Removed the old combined edit form and single Edit button from the heading row
- Added .card-editable and .card-edit-btn CSS for absolute-positioned edit button inside cards
