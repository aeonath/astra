# Changelog — 2026-03-01 09:00

## Added: Search functionality

- Search box in the top navigation bar (to the left of the username, logged-in users only)
- GET /bugs/search route searches bugs, features, and todos by title or description
- Results displayed in a table with type, priority, status, project, and date columns
- Clickable rows navigate to the item detail page
- Results capped at 100 items, ordered by most recently updated
