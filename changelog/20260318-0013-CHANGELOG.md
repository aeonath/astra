# Changelog — 2026-03-18

## Changed
- **Revised project visibility terminology:**
  - **Active** = shown on the Projects Page (replaces the old "internally visible")
  - **Publicly Visible** = shown on the external landing page (unchanged behavior)
  - **Projects Summary** = shows all non-archived projects (no longer requires active flag)
  - **Manage Projects** = shows all projects including archived
- Removed the "Internally visible" checkbox and all references from routes, views, and JS.
- The "Publicly visible" checkbox no longer depends on "Internally visible".
- Updated visibility badges in the Manage Projects table to show Active/Public/Hidden.
