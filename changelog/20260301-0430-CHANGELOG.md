# Changelog — 2026-03-01 04:30

## Editable bug title/description, project default assignee

### Bug detail view
- Title and description are now editable fields in the sidebar form instead of hidden inputs.
- The description section is always visible, showing "No description provided." when empty.

### Project default assignee
- Added `default_assignee_id` column to the projects table (migration 006).
- Admin project creation form now includes a default assignee dropdown.
- The default assignee column is shown in the admin projects table.
- When creating a new bug/feature/todo, the assignee dropdown is pre-selected to the project's default assignee.
- If no assignee is explicitly chosen, the project default is used as a fallback.
