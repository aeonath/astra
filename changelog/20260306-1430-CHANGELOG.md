# Changelog — 2026-03-06 14:30

## Add persistent project tabs in navbar

- Opening a project page creates a tab in the navbar next to the Projects button
- Viewing a bug also opens/highlights its project tab
- Tabs persist in localStorage until manually closed with the × button
- Closing a tab while on that project redirects to /projects
- Active tab is highlighted when on the project page or any of its bugs
- Projects nav link no longer marks active for sub-pages (avoids conflict with tabs)
- Tabs scroll horizontally if many are open
