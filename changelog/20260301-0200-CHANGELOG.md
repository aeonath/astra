# Changelog — 2026-03-01 02:00

## Fix datetime("now") SQL quoting bug

SQLite double quotes are for identifiers, not strings. `datetime("now")` was being rejected by newer SQLite versions with "no such column: now". Changed all occurrences to `datetime('now')` with proper single quotes.

Affected files: `src/routes/admin.js`, `src/db/reset-password.js`. This also fixes the 500 error on the web UI password reset and admin toggle actions.
