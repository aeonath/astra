# Changelog — 2026-03-18

## Changed
- Renamed "Keywords" to "Tags" throughout the project card system.
- Tags are now stored comma-separated and displayed as individual tag pills.
- Replaced plain text input with a GitHub-style tag input component: type a tag and press Enter or comma to add it as a pill, Backspace to remove the last tag, and click the x to remove any tag.
- Migration 023 renames the `keywords` column to `tags` in the projects table.
