# Changelog — 2026-03-06 12:45

## Add comment edit and admin delete features

- Added Edit button inside each comment card; clicking it shows an inline edit form and hides the display text
- Cancel button restores the display without saving
- Added POST /bugs/:id/comment/:commentId/edit route to save edited comment content
- Added Delete button visible only to admins; clicking prompts for confirmation before deleting
- Added POST /bugs/:id/comment/:commentId/delete route (admin-only, returns 403 if not admin)
- Added .comment-actions CSS for the edit/delete button group in the comment header
