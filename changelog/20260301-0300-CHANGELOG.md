# Changelog — 2026-03-01 03:00

## Fix edit user click handler blocked by CSP

Moved the inline JavaScript from the users template to an external file (`public/js/admin-users.js`) and replaced inline `onclick` handlers with `data-*` attributes and event listeners. This fixes the edit-on-click feature being silently blocked by Helmet's Content Security Policy (`scriptSrc: ["'self'"]`).
