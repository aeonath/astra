# Changelog — 2026-03-18

## Added
- **Project Card system**: Each project now has card details (nickname, scope, purpose, status, keywords) modeled after STUDIO_PROJECTS.md format.
- **Projects Summary page** (`/projects/summary`): Shows project cards with full details followed by an overview table with bug/feature/todo counts. Cards are editable inline.
- **Project Card modal** on project detail pages: "Project Card" button in the header opens a modal to edit card details.
- Migration 022 adds `nickname`, `scope`, `purpose`, `project_status`, and `keywords` columns to the projects table.
- POST `/projects/summary/:id/card` endpoint for saving project card data with redirect support for both pages.
- CSS for summary cards, keyword pills, and wider modal variant.
