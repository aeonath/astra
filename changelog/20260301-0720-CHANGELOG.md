# Changelog — 2026-03-01 07:20

## Added: User-level submissions management

- New `can_manage_submissions` flag on user accounts (migration 013)
- Admins can grant/revoke submissions access per user via the Manage Users page
- Users with the permission see a "Submissions" link in the navbar
- User-facing `/submissions` route with the same review/dismiss workflow as the admin inbox
- Admins always have submissions access automatically
