# Changelog — 2026-03-06 15:11

## Import button for public submissions

- Migration 019: adds imported_bug_id column to public_submissions to track import state
- Admin POST /submissions/:id/import: creates a bug or FR from the submission (medium priority, project default assignee), adds a first comment containing the submitter name, email, original title and description, marks submission as reviewed and sets imported_bug_id, redirects to the new bug
- Submissions view: shows Import button (blue) when not yet imported, replaced by a View link once imported — prevents double import
