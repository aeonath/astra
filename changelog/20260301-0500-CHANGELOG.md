# Changelog — 2026-03-01 05:00

## Add project categories with admin-controlled display order

### New: Categories
- New `categories` table with name and sort_order (migration 007).
- Four default categories seeded: Apps, Games, Websites, Extensions.
- Projects now have an optional `category_id` foreign key.

### Admin
- New "Manage Categories" page at `/admin/categories` — add, reorder (up/down), and delete categories.
- Categories can only be deleted if no projects use them.
- Project creation form now includes a category dropdown.
- Projects table shows the assigned category.
- Dashboard has a "Manage Categories" link.

### Public projects page
- Projects are now grouped by category with section headings, ordered by the admin-defined sort order.
- Uncategorized projects appear at the end under "Other".
