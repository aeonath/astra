# Changelog — 2026-03-01 07:00

## Added: Public project access site setting

- New site setting `public_project_access` controls whether public (not logged in) visitors can click into individual project pages
- When enabled (default): visitors can click project cards on the landing page to view project details
- When disabled: visitors see the project list with stats on the landing page but project cards are not clickable and direct URL access redirects to the home page
- Admin settings page has a new checkbox to toggle this setting
- Migration 011 seeds the new setting with a default value of enabled
