# Changelog — 2026-03-18

## Added
- Drag-and-drop reordering of project cards on the Projects Summary page. Drag a card by the handle on the left to change its priority order. New order is auto-saved to the server.
- Migration 026 adds `summary_sort_order` column to projects, initialized alphabetically for existing active projects.
- POST `/projects/summary/reorder` endpoint accepts a JSON array of project IDs to persist the new order.
