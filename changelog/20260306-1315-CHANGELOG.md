# Changelog — 2026-03-06 13:15

## Closed-status UI lockdown for bug/feature/todo detail page

- When status is 'closed', Description and Additional Notes headings and text dim to #cccccc
- Edit buttons for both cards are hidden when closed
- Priority and Assignee selects are disabled when closed (can't be changed)
- All of the above update dynamically when the status select is changed (before saving)
- Server-side: when setting status to 'closed', existing priority and assignee are always preserved regardless of submitted values
