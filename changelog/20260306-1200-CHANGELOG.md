# Changelog — 2026-03-06 12:00

## User Profile Page

Added a profile page allowing logged-in users to update their display name and change their password.

### Changes
- `src/routes/auth.js` — Added `GET /profile`, `POST /profile/display-name`, and `POST /profile/password` routes (all protected by `requireLogin`)
- `src/views/auth/profile.ejs` — New profile page with two forms: one for display name, one for password change (requires current password, validates new password match and minimum 8 characters)
- `src/views/partials/header.ejs` — Username in nav bar is now a link to `/profile`
- `public/css/style.css` — Added `.profile-grid` two-column layout, `.card-title` style, and updated `.user-name` to render as a styled link
