# Changelog — 2026-03-06 15:12

## Add reCAPTCHA v2 to public submission form

- CSP: added Google reCAPTCHA script and frame domains to scriptSrc and frameSrc
- locals.js: injects recaptchaSiteKey from RECAPTCHA_SITE_KEY env var into all views
- public-submit.ejs: renders g-recaptcha widget (dark theme) and loads reCAPTCHA API script when sitekey is configured
- public-submit-recaptcha.js: client-side validation — blocks form submit if reCAPTCHA not completed
- projects.js POST /submit: server-side verification via Google siteverify API using RECAPTCHA_SECRET_KEY; skipped if key not set
- .env / .env.example: added RECAPTCHA_SITE_KEY and RECAPTCHA_SECRET_KEY (same sitekey as miranova.studio)
- CSS: recaptcha-container and recaptcha-error styles added
- Widget and verification are fully optional — if keys not set, the form works without reCAPTCHA (useful for dev)
