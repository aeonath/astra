# Changelog — 2026-03-01 05:15

## Click-to-edit projects in admin panel

Added the same click-to-edit pattern used on the users page to the projects admin page. Clicking a project row populates the form with the project's current details (description, category, default assignee, visibility) for editing. Name is read-only after creation. A Cancel button resets back to the "Add Project" form.
