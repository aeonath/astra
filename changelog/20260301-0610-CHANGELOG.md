# Changelog — 2026-03-01 06:10

## Custom display IDs and improved bug detail layout

- **Display IDs for bugs and features:** Bugs now show as `BUG-001`, `BUG-002` etc. Feature requests show as `REQ-001`, `REQ-002` etc. IDs are sequential per project and type, stored in a `display_number` column (migration 009 backfills existing records).
- **Todo numbering:** Open todos are numbered 1, 2, 3 dynamically based on position. Closed todos have no number. Status column removed from todo list since the tab already indicates open/closed.
- **Bug detail layout:** The title now sits inside the two-column grid (left of the details sidebar) instead of spanning the full page width. Breadcrumb stays above the grid. The edit button sits inline with the title.
- **ID in sidebar metadata:** The detail sidebar now shows the display ID (e.g. BUG-001) instead of the type label.
