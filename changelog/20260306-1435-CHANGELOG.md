# Changelog — 2026-03-06 14:35

## Fix project tabs blocked by Content Security Policy

- Helmet's scriptSrc: ["'self'"] blocks all inline scripts, which silently killed the __openProjectTab() calls
- Replaced inline script approach with a hidden <div id="current-project" data-slug data-name> element
- project-tabs.js reads the data attributes in DOMContentLoaded — no inline JS needed
- Removed the now-unused window.__openProjectTab export
