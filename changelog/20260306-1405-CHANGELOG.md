# Changelog — 2026-03-06 14:05

## Add file attachments to bug view

- Added FILES_DIR env var to .env and .env.example (absolute path for file storage)
- Added migration 016: bug_files table (bug_id, filename, size, uploaded_by, uploaded_at, UNIQUE per bug+filename)
- Files stored at {FILES_DIR}/{project_slug}/{bug_id}/{filename}
- POST /bugs/:id/files — uploads a file; errors if duplicate filename already attached to that bug
- GET /bugs/:id/files/:filename — downloads a file; errors if file record exists in DB but file is missing from disk
- Attachments section visible only on bug-type items (not features or todos)
- Installed multer for multipart/form-data handling (memory storage, written to disk after duplicate check)
